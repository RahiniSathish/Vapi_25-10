# ---------------------------------------------------------------
# KEY DIFFERENCES
# ---------------------------------------------------------------

VAPI_PUBLIC_KEY (pk_...):
  ✅ Use: Frontend/Client-side JavaScript
  ✅ Safe to expose in browser
  ✅ Purpose: Start calls, connect to assistant
  ❌ Cannot: Access MCP server

VAPI_PRIVATE_KEY (sk_...):
  ✅ Use: Backend/Server-side Python
  ⚠️ NEVER expose in frontend
  ✅ Purpose: MCP connection, server API access
  ✅ Required for: MCP server real-time communication

# ---------------------------------------------------------------
# HOW TO GET VAPI_PRIVATE_KEY
# ---------------------------------------------------------------

1. Go to: https://dashboard.vapi.ai
2. Navigate to: Settings → API Keys
3. Look for: "Private API Key" or "Server Token"
4. Copy the key that starts with: sk_...
5. Paste it in your .env file as VAPI_PRIVATE_KEY
